# Diary App

A full-stack journaling application built with React, Express, and PostgreSQL.

## Features

- **Daily Journal Entries** — Record your thoughts, gratitude, and reflections.
- **Task Tracking** — Create and mark tasks as complete.
- **Mood & Energy Logging** — Track your well-being with mood and energy levels.
- **Health Metrics** — Log sleep hours, workouts, meditation, and healthy eating.
- **Statistics Dashboard** — View your progress with visual stats (mood trends, completion rates, sleep duration).
- **Calendar View** — See which days you've journaled with highlighted entries.
- **PDF Export** — Generate and download PDF reports of your journal entries.
- **User Authentication** — Sign up and log in to keep your entries private.

## Tech Stack

- **Frontend**: React, React Router, Vite
- **Backend**: Express.js, Node.js
- **Database**: PostgreSQL
- **PDF Generation**: @react-pdf/renderer
- **Styling**: CSS
